How this simulation works

Each simulation plays out every team's remaining games. Opponents are sampled from a division-weighted schedule (2/3 divisional, 1/3 cross-division), matching how the actual MMS schedule generally distributes.

Each simulated game uses a logistic function on the two teams' iterative Power Index ratings to compute win probability, with small random variance added to reflect softball's inherent chaos. The projections should be interpreted as "based on how each team has looked so far, here are the finish distributions" — not a prediction of any specific game.

Adjust the "Games per team total" if the actual season length differs from the default of 21. Rerun the simulation any time.
